# Vault Omar de Mogador

> [!info] Bienvenue
> Ce vault Obsidian contient l'ensemble de la documentation stratégique du projet **Omar de Mogador** — identité de marque, équipe d'avatars IA, étude de marché, sourcing fournisseurs et plan d'action.

---

## Comment ouvrir ce vault

### 1. Installer Obsidian
Téléchargement gratuit : https://obsidian.md

### 2. Ouvrir le dossier comme vault
- Lancez Obsidian
- Cliquez sur *Ouvrir un coffre* (ou *Open vault*)
- Sélectionnez le dossier `omar-de-mogador-vault`
- Validez

### 3. Commencer par
Ouvrez le fichier **`00-Index/MOC - Omar de Mogador.md`** — il sert d'index principal et de point d'entrée vers tous les autres documents.

## Fonctionnalités Obsidian utilisées

- **Liens internes (wikilinks)** — `[[Nom du fichier]]` pour naviguer entre les notes
- **Tags** — `#sourcing`, `#avatar`, etc. pour filtrer transversalement
- **Frontmatter YAML** — métadonnées en début de chaque fichier
- **Callouts** — encadrés colorés `> [!info]`, `> [!warning]`, etc.
- **Diagrammes Mermaid** — pour les schémas et plannings
- **Tableaux Markdown** — pour les comparaisons
