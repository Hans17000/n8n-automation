# Conventions — Le Cabinet d'Iris

## Structure des dossiers

```
01-Plan/          → Vision, branding, fournisseurs, roadmap, catalogue
02-Dev/           → Specs, scripts, PrestaShop, automatisation
03-SEO/           → Stratégie, catégories SEO, pages piliers
04-Produits/      → Fiches produit, descriptions, analyses
05-Design/        → Identité visuelle, moodboard, customisation thème
06-IA/            → Prompts, workflows, conversations
```

## Nommage

- Fichiers : `Type - Sujet.md` (ex: `Template - Fiche Produit.md`)
- Dossiers : numérotés pour l'ordre logique
- Références produit : `CDI-{id_fournisseur}`

## Liens

- Le MOC (`MOC - Le Cabinet d'Iris.md`) est le point d'entrée
- Les `[[liens wiki]]` connectent les notes entre elles
- Le document complet de référence est [[Le-Cabinet-dIris-Univers-Complet]]
