# Prompt — Voix d'Iris (descriptions produit)

> À développer quand la phase descriptions démarrera.

## Principe

Chaque description de produit doit être écrite dans la voix d'Iris — littéraire, suggestive, jamais crue. Voir [[Branding]] pour les exemples de ton.

## Structure type

1. Phrase d'accroche narrative (évocation sensorielle)
2. Caractéristiques produit intégrées dans la prose
3. Détails techniques (matière, tailles, couleurs)

## Contraintes

- Jamais de vocabulaire médical ou porno
- Pas de points d'exclamation
- Pas de superlatifs commerciaux (« incroyable », « révolutionnaire »)
